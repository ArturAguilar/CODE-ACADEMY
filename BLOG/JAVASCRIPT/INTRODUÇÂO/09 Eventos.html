Eventos em JavaScript

Eventos são ações que ocorrem na página, como cliques, pressionamentos de tecla ou carregamento de elementos. O JavaScript pode ser usado para capturar e responder a esses eventos.

O Que São Eventos?

Eventos permitem que o JavaScript reaja às interações do usuário. Alguns exemplos comuns incluem:

✅ onclick → Quando um botão é clicado.
✅ onmouseover → Quando o mouse passa sobre um elemento.
✅ onkeydown → Quando uma tecla é pressionada.
✅ onload → Quando a página termina de carregar.

Como Lidar com Eventos

Existem três formas principais de atribuir eventos a elementos no JavaScript:

Atributo HTML (não recomendado)

Podemos adicionar um evento diretamente no HTML, mas isso não é recomendado porque mistura HTML com JavaScript.

<button onclick="alert('Botão clicado!')">Clique aqui</button>


Manipulador de Eventos no JavaScript (Forma Recomendada)

A melhor prática é usar o addEventListener(), pois permite maior controle sobre os eventos.

<button id="meuBotao">Clique aqui</button>

<script>
document.getElementById("meuBotao").addEventListener("click", function() {
alert("Botão clicado!");
});
</script>

Aqui estamos:
✅ Pegando o botão pelo id.
✅ Adicionando um evento de clique.
✅ Executando a função quando o botão for clicado.


Propriedade onevent (Alternativa)

Também podemos definir eventos diretamente no JavaScript com onevent.

<button id="meuBotao">Clique aqui</button>

<script>
document.getElementById("meuBotao").onclick = function() {
alert("Botão clicado!");
};
</script>
