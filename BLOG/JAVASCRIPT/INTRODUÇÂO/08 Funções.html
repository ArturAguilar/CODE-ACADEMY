Funções em JavaScript

Funções são blocos de código reutilizáveis que podem ser chamados para executar uma determinada tarefa. No JavaScript, existem diferentes formas de declarar funções, incluindo as funções tradicionais e as arrow functions.

Criando e Chamando Funções Tradicionais

Uma função no JavaScript é definida com a palavra-chave function, seguida por um nome, parênteses () e um bloco de código {}.

Sintaxe

function nomeDaFuncao(param1, param2) {
// Código a ser executado
return resultado;
}


Exemplo Simples

function somar(a, b) {
return a + b;
}

console.log(somar(3, 5)); // Saída: 8

  • a e b são parâmetros.
  • return devolve o resultado da soma para quem chamou a função

Chamando Funções

let resultado = somar(10, 20); console.log(resultado); // 30

Funções sem Parâmetros

Se uma função não precisar de parâmetros, basta omitir os valores dentro dos parênteses.

function mensagem() {
return "Olá, mundo!";
}

console.log(mensagem()); // "Olá, mundo!"

Funções com Parâmetros Padrão

Se nenhum valor for passado para um parâmetro, podemos definir um valor padrão.

function saudacao(nome = "visitante") {
return "Olá, " + nome + "!";
}

console.log(saudacao("Maria")); // "Olá, Maria!"
console.log(saudacao()); // "Olá, visitante!"

Funções Anônimas

As funções anônimas não possuem nome e são geralmente usadas dentro de variáveis.

let somar = function(a, b) {
return a + b;
};

console.log(somar(5, 10)); // 15

Arrow Functions

As arrow functions (funções de seta) são uma forma mais curta de escrever funções.

Sintaxe

let nomeDaFuncao = (param1, param2) => {
// Código a ser executado
return resultado;
};

Se houver apenas uma linha, podemos omitir {} e return.


Exemplo

// Função tradicional
function dobrar(x) {
return x * 2;
}

// Mesma função com arrow function
const dobrar = x => x * 2;

console.log(dobrar(4)); // 8

Se a função não tiver parâmetros, usamos () vazios:

let mensagem = () => "Olá, mundo!";
console.log(mensagem()); // "Olá, mundo!"

Diferença entre Função Normal e Arrow Function

Arrow functions não possuem seu próprio this, ou seja, elas herdam o this do contexto onde foram criadas.

Exemplo com this

const pessoa = {
nome: "Artur",
idade: 18,
falar: function() {
console.log(`Meu nome é ${this.nome}`);
}
};

pessoa.falar(); // "Meu nome é Artur"

Se usarmos uma arrow function dentro do objeto, this não se referirá ao objeto.

const pessoa = {
nome: "Artur",
idade: 18,
falar: () => {
console.log(`Meu nome é ${this.nome}`);
}
};

pessoa.falar(); // "Meu nome é undefined"

⚠️ Use funções tradicionais dentro de objetos para evitar problemas com this.

Função como Parâmetro (Callback)

Funções podem ser passadas como argumento para outras funções.

function processar(numero, funcao) {
return funcao(numero);
}

console.log(processar(5, dobrar)); // 10
console.log(processar(4, x => x ** 2)); // 16
