Elemento canvas no HTML

O canvas é uma área de desenho que permite criar gráficos, animações e outras imagens dinâmicas usando JavaScript.

Criando um Canvas Simples

Para adicionar um canvas, basta definir seu id, largura (width) e altura (height).

<canvas id="meuCanvas" width="400" height="200" style="border:1px solid #000;">
Seu navegador não suporta o elemento canvas.
</canvas>

✅ O canvas é exibido como um retângulo de 400x200px com borda preta.

Acessando o Canvas via JavaScript

Para desenhar no canvas, usamos JavaScript para acessar o contexto de renderização.

<script>
let canvas = document.getElementById("meuCanvas");
let ctx = canvas.getContext("2d"); // Obtém o contexto 2D
</script>

✅ O método getContext("2d") permite desenhar no canvas.

Desenhando Formas

Retângulo Cheio (fillRect)

ctx.fillStyle = "#d70000"; // Cor vermelha
ctx.fillRect(50, 50, 100, 50); // (x, y, largura, altura)

✅ Desenha um retângulo vermelho de 100x50px na posição (50,50).

Retângulo com Borda (strokeRect)

ctx.strokeStyle = "#000"; // Cor preta
ctx.strokeRect(50, 50, 100, 50);

✅ Apenas o contorno do retângulo será visível.

Apagar uma Área (clearRect)

ctx.clearRect(60, 60, 30, 30);

✅ Apaga uma área de 30x30px na posição (60,60).

Desenhando Linhas

Para criar uma linha, usamos beginPath(), moveTo(x, y) e lineTo(x, y).

ctx.beginPath();
ctx.moveTo(20, 20); // Ponto inicial
ctx.lineTo(150, 100); // Ponto final
ctx.stroke(); // Desenha a linha

✅ Cria uma linha reta de (20,20) a (150,100).

Desenhando Círculos e Arcos

Arco/Círculo (arc)

ctx.beginPath();
ctx.arc(100, 100, 40, 0, 2 * Math.PI); // (x, y, raio, ângulo inicial, ângulo final)
ctx.fillStyle = "blue";
ctx.fill();

✅ Desenha um círculo azul de raio 40px no ponto (100,100).

Desenhando Texto

Texto Preenchido (fillText)

ctx.font = "20px Arial";
ctx.fillStyle = "green";
ctx.fillText("Olá, Canvas!", 50, 50);

✅ Exibe "Olá, Canvas!" em verde na posição (50,50).

Texto com Contorno (strokeText)

ctx.strokeStyle = "red";
ctx.strokeText("Texto com borda", 50, 80);

✅ Desenha texto com borda vermelha.

Adicionando Imagens ao Canvas

Podemos carregar imagens dentro do canvas.

let img = new Image();
img.src = "minha_imagem.jpg";
img.onload = function () {
ctx.drawImage(img, 50, 50, 100, 100); // (imagem, x, y, largura, altura)
};

✅ Insere uma imagem redimensionada de 100x100px na posição (50,50).
