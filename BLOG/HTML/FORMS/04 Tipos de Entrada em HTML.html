Tipos de Entrada em HTML

O elemento input suporta vários tipos diferentes, cada um projetado para capturar um tipo específico de dado do usuário.

type="text" (Texto)

Cria um campo de entrada para texto simples.

<input type="text" name="nome" placeholder="Digite seu nome">

type="password" (Senha)

Cria um campo de senha, ocultando os caracteres digitados.

<input type="password" name="senha">

type="email" (E-mail)

Aceita apenas endereços de e-mail válidos.

<input type="email" name="email">

type="number" (Número)

Aceita apenas valores numéricos.

<input type="number" name="idade" min="1" max="100">

type="tel" (Telefone)

Aceita um número de telefone.

<input type="tel" name="telefone" placeholder="(xx) xxxxx-xxxx">

type="url" (URL)

Aceita apenas URLs válidas.

<input type="url" name="site" placeholder="https://exemplo.com">

type="search" (Busca)

Cria um campo de pesquisa.

<input type="search" name="pesquisa" placeholder="Buscar...">

type="date" (Data)

Permite a seleção de uma data no formato YYYY-MM-DD

<input type="date" name="nascimento">

type="datetime-local" (Data e Hora)

Permite selecionar data e hora.

<input type="datetime-local" name="evento">

type="month" (Mês e Ano)

Seleciona apenas o mês e o ano.

<input type="month" name="mes">

type="week" (Semana do Ano)

Permite selecionar uma semana específica.

<input type="week" name="semana">

type="time" (Hora)

Permite selecionar uma hora específica.

<input type="time" name="horario">

type="color" (Cor)

Abre um seletor de cores.

<input type="color" name="cor_favorita">

type="file" (Arquivo)

Permite que o usuário envie arquivos.

<input type="file" name="arquivo">

type="checkbox" (Caixa de Seleção)

Cria uma opção que pode ser marcada ou desmarcada.

<input type="checkbox" name="aceitar" value="sim"> Aceito os termos

type="radio" (Botão de Opção)

Permite selecionar apenas uma opção dentro de um grupo.

<input type="radio" name="genero" value="masculino"> Masculino
<input type="radio" name="genero" value="feminino"> Feminino

type="range" (Controle Deslizante)

Permite escolher um valor dentro de um intervalo.

<input type="range" name="volume" min="0" max="100">

type="hidden" (Campo Oculto)

Armazena dados sem exibi-los ao usuário.

<input type="hidden" name="id_usuario" value="12345">

type="submit" (Botão de Envio)

Cria um botão para enviar o formulário.

<input type="submit" value="Enviar">

type="reset" (Botão de Reset)

Redefine os campos do formulário.

<input type="reset" value="Limpar">

type="button" (Botão Comum)

Cria um botão que pode ser usado com JavaScript.

<input type="button" value="Clique Aqui" onclick="alert('Botão clicado!')">
