O que são formulários em HTML?

Os formulários são usados em páginas web para coletar dados do usuário, como nome, e-mail, senha, entre outros. Eles são essenciais para interação, como fazer login, enviar dados ou fazer pesquisas.

Estrutura Básica de um Formulário

Para criar um formulário, usamos a tag form e vários tipos de campos de entrada, como input, textarea, button.

Exemplo de um formulário simples:

<form action="submit_form.asp" method="post">
<label for="fname">Primeiro Nome:</label>
<input type="text" id="fname" name="fname">
<br>

<label for="email">Email:</label>
<input type="email" id="email" name="email">
<br>

<input type="submit" value="Enviar">
</form>

Atributos Importantes do Formulário

  • action: Define a URL para onde os dados do formulário serão enviados. Exemplo: action="submit_form.asp".
  • method: Define o método de envio dos dados. Pode ser GET ou POST.
    • GET: Os dados são anexados à URL. Usado para enviar pequenas quantidades de dados.
    • POST: Os dados são enviados no corpo da requisição. Usado para enviar grandes quantidades de dados.

Tipos de Campos de Entrada

Campo de texto:

Usado para texto simples.

<input type="text" id="fname" name="fname">

Campo de senha:

Usado para senha, com os caracteres ocultos.

<input type="password" id="psw" name="psw">

Campo de e-mail:

Usado para validar e-mails.

<input type="email" id="email" name="email">

Área de texto:

Usado para textos mais longos.

<textarea id="message" name="message"></textarea>

Botão de envio:

Envia o formulário.

<input type="submit" value="Enviar">

Exemplo Completo de Formulário

<form action="submit_form.asp" method="post">

Validação de Formulário

A validação é uma técnica para garantir que o usuário preencheu corretamente os campos antes de enviar o formulário. Isso pode ser feito com atributos HTML como:

  • required: O campo não pode ser vazio.
  • minlength: Define o número mínimo de caracteres.
  • maxlength: Define o número máximo de caracteres.

Exemplo:

<input type="text" id="fname" name="fname" required>
