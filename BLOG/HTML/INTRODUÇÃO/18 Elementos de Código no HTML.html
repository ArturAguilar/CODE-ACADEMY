Elementos de Código no HTML

  • Os elementos de código no HTML são usados para representar texto relacionado a código de programação, como trechos de código, variáveis, comandos, ou saídas do console.

Elementos Principais

code

  • Define um trecho de código.

<p>A função <code>alert()</code> exibe uma mensagem para o usuário.</p>

pre

  • Mantém o formato do texto, como quebras de linha e espaços.
  • Ideal para exibir blocos de código ou texto pré-formatado.

<pre>
function myFunction() {
alert("Hello World!");
}
</pre>

samp

  • Representa uma saída de computador.

<p>Saída esperada: <samp>Erro 404: Página não encontrada</samp></p>

kbd

  • Representa entrada de teclado (algo que o usuário deve digitar).

<p>Pressione <kbd>Ctrl</kbd> + <kbd>S</kbd> para salvar.</p>

var

  • Define variáveis em código ou matemática.

<p>A fórmula da área de um círculo é <var>πr²</var>.</p>

output

  • Representa o resultado de um cálculo ou script.

<label for="number">Digite um número:</label>
<input type="number" id="number" oninput="calculate()"> <output id="result"></output>

<script>
function calculate() {
const num = document.getElementById("number").value; document.getElementById("result").textContent = num * 2;
}
</script>
