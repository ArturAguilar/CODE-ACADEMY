Elementos de Citação em HTML

Os elementos de citação são usados para exibir textos citados, como trechos de outros autores, ou para atribuir referências.

Principais Tags de Citação

Citações Curtas (<q>)

Usada para citar trechos curtos.

O navegador adiciona automaticamente aspas ao redor do texto.

<p>Albert Einstein disse: <q>A imaginação é mais importante que o conhecimento.</q></p>

Citações Longas (<blockquote>)

Usada para trechos longos de citação.

Normalmente exibida como texto destacado ou em um bloco separado.

Pode incluir o atributo cite para referenciar a fonte.

<blockquote cite="https://www.exemplo.com">
Este é um exemplo de citação longa em HTML.
</blockquote>

Atribuição de Fonte (<cite>)

Define a fonte de uma obra (livro, site, artigo, etc.).

Geralmente exibida em itálico.

<p>Esta frase é de <cite>O Pequeno Príncipe</cite>.</p>

Definições de Abreviação (<abbr>)

Indica uma abreviação ou sigla.

Use o atributo title para fornecer o significado completo.

<p>A sigla <abbr title="World Health Organization">WHO</abbr> é conhecida mundialmente.</p>

Exemplo Completo

<blockquote cite="https://www.brainyquote.com">
<p>Albert Einstein disse: <q>A vida é como andar de bicicleta. Para manter o equilíbrio, você deve continuar se movendo.</q></p>
</blockquote>
