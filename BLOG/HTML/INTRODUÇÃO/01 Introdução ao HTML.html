Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web.
Utiliza elementos que definem a estrutura e o conteúdo de uma página.

Elementos HTML

Um elemento HTML é representado por uma tag (ex.: <tag>).br Exemplos:
<h1>: Título principal.
<p>: Parágrafo.
<a>: Link.

Estrutura Básica de um Documento HTML

<!DOCTYPE html>
<html>
<head>
<title>Meu Título</title>
</head>
<body>

<h1>Bem-vindo ao HTML</h1>
<p>Este é um parágrafo.</p>

</body>
</html>


Principais Características

Facilidade de Uso: Simples de aprender e usar.

Compatibilidade: Suportado por todos os navegadores modernos.

Flexibilidade: Usado em conjunto com CSS e JavaScript para criar designs dinâmicos e interativos.

Outros Conceitos

HTML é uma linguagem de marcação, não uma linguagem de programação.

Cada elemento HTML pode ter atributos (ex.: id, class) que adicionam funcionalidade ou estilo.
