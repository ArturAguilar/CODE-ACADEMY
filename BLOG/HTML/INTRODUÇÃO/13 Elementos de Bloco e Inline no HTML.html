Elementos de Bloco e Inline no HTML

Elementos de Bloco (Block Elements)

  • Definição: Elementos de bloco sempre começam em uma nova linha e ocupam toda a largura disponível (extensão máxima do elemento pai).
  • São usados principalmente para estruturar o layout de uma página.

Exemplos de Elementos de Bloco Comuns:

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <p>, <div>, <header>, <footer>, <section>, <article>, <aside>, <main>, <nav>, <form>

Elementos Inline (Inline Elements)

  • Definição: Elementos inline não iniciam em uma nova linha. Eles ocupam apenas a largura necessária e podem ser usados dentro de elementos de bloco.
  • São usados principalmente para estilizar ou formatar partes específicas do conteúdo.

Exemplos de Elementos Inline Comuns:

<span>, <a>, <strong>, <em>, <b>, <i>, <u>, <img>, <br>, <small>

Diferenças Entre Bloco e Inline

Característica Elementos de Bloco Elementos Inline
Início de nova linha Sempre inicia em uma nova linha Não inicia em uma nova linha
Ocupação de largura Ocupa toda a largura disponível (pai) Ocupa apenas o espaço necessário
Conteúdo permitido Pode conter outros elementos de bloco ou inline Apenas outros elementos inline

Combinação de Bloco e Inline

<p> Este é um texto com <strong>destaque</strong> e um <a href="#">link</a>. </p>
