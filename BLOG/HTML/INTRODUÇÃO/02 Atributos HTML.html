Atributos HTML

O que são atributos HTML?

Atributos fornecem informações adicionais sobre elementos HTML.
São especificados dentro da tag de abertura de um elemento.

Regras Gerais

Sempre escritos como nome="valor".
O nome do atributo e seu valor são case-insensitive (não diferenciam maiúsculas de minúsculas).
Devem ser escritos em tags de abertura, nunca em tags de fechamento.

Exemplos Comuns de Atributos

Atributo href (Links):

Define o destino de um link.

<a href="https://www.w3schools.com">Visite a W3Schools</a>

Atributo src (Imagens):

Define o caminho da fonte de uma imagem.

<img src="imagem.jpg" alt="Descrição da Imagem">

Atributo alt (Texto Alternativo):

Fornece texto alternativo caso a imagem não carregue.

<img src="imagem.jpg" alt="Descrição da Imagem">

Atributo style (Estilo Inline):

Adiciona estilos CSS diretamente ao elemento.

<p style="color:red;">Este é um parágrafo vermelho.</p>

Atributo id (Identificação):

Dá um identificador único ao elemento.

<h1 id="titulo">Título Principal</h1>

Atributo class (Classe):

Associa o elemento a uma classe CSS.

<p class="texto-destaque">Este é um parágrafo destacado.</p>

Atributos Globais

São aplicáveis a quase todos os elementos HTML:
- id
- class
- style
- title (exibe texto ao passar o mouse sobre o elemento)
- data-* (atributos personalizados para armazenar dados)

Exemplo de um atributo global:

<div id="meuDiv" title="Passe o mouse aqui!">Conteúdo</div>
