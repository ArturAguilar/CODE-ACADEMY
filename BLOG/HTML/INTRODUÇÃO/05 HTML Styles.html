HTML Styles

O que são estilos HTML?

Estilos em HTML são usados para adicionar design e personalização visual aos elementos.

O atributo style permite aplicar regras de CSS diretamente em um elemento HTML.

Atributo style

O atributo style define estilos inline para elementos.

Sintaxe: style="propriedade: valor;".

<p style="color:blue;">Este é um parágrafo azul.</p>

Propriedades Comuns de Estilo

Cor do texto (color):

Define a cor do texto.

<p style="color:red;">Texto vermelho</p>

Cor de fundo (background-color):

Define a cor de fundo do elemento.

<p style="background-color:yellow;">Texto com fundo amarelo</p>

Fonte (font-family):

Define o tipo de fonte do texto.

<p style="font-family:Arial;">Texto com fonte Arial</p>

Tamanho da fonte (font-size):

Define o tamanho da fonte do texto.

<p style="font-size:20px;">Texto com tamanho 20px</p>

Alinhamento do texto (text-align):

Alinha o texto dentro do elemento.

<p style="text-align:center;">Texto centralizado</p>

Margens e preenchimento (margin e padding):

Define o espaço ao redor do elemento.

<p style="margin:10px;">Texto com margem de 10px</p>

Boas Práticas

Evitar Estilos Inline:

Para manter o código limpo e fácil de manter, use estilos em arquivos CSS externos ou internos em vez de inline.

Exemplo de CSS externo:

<head>
.code p{ background-color: #e6e4e4; border-radius: 10px; min-width: 300px; max-width: 450px; font-family: 'Fira Code', sans-serif; margin: auto; padding: 10px; overflow: hidden; } <link rel="stylesheet" href="styles.css">
</head>

Separar HTML e CSS:

HTML é para estrutura, CSS é para estilos.
