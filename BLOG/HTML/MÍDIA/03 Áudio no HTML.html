Áudio no HTML

O HTML5 permite incorporar áudio diretamente na página usando a tag audio.

O Elemento audio

A tag audio adiciona áudio ao site sem precisar de plugins externos.

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
Seu navegador não suporta áudio.
</audio>

✅ O navegador escolhe automaticamente a melhor fonte de áudio.

✅ O atributo controls adiciona botões de play, pause e volume.

Formatos de Áudio Suportados

Nem todos os navegadores suportam os mesmos formatos.

Formato Tipo MIME Navegadores Suportados
MP3 audio/mpeg Chrome, Edge, Safari, Firefox, Opera
WAV audio/wav Chrome, Edge, Safari, Firefox
OGG audio/ogg Chrome, Firefox, Opera

💡 Dica: Use múltiplas fontes para melhor compatibilidade.

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
<source src="audio.ogg" type="audio/ogg">
Seu navegador não suporta áudio.
</audio>

Atributos da Tag audio

Atributo Descrição
controls Exibe os controles do player
autoplay O áudio inicia automaticamente
loop O áudio repete continuamente
muted Começa sem som

Exemplo com todos os atributos:

<audio controls autoplay loop muted>
<source src="audio.mp3" type="audio/mpeg">
</audio>

✅ O áudio começa automaticamente, sem som, e repete indefinidamente.

Adicionando Áudio de Background (Evite!)

O atributo autoplay pode ser usado para tocar áudio de fundo, mas muitos navegadores bloqueiam reprodução automática.

Se precisar de som ambiente, informe o usuário e adicione um botão manual.

<button onclick="document.getElementById('som').play()">
Tocar música
</button>

<audio id="som">
<source src="audio.mp3" type="audio/mpeg">
</audio>

Incorporando Áudio Externo

Para Spotify ou SoundCloud, use iframe:

<iframe src="https://open.spotify.com/embed/track/ID_DA_MUSICA"
width="300" height="80" frameborder="0" allow="encrypted-media">
</iframe>

✅ Substitua "ID_DA_MUSICA" pelo código da música no Spotify.
