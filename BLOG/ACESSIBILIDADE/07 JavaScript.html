JavaScript

No JavaScript, o foco no teclado pode ser manipulado para melhorar a acessibilidade e a experiência do usuário. Aqui estão alguns pontos importantes sobre como lidar com o foco no teclado usando JavaScript:

Métodos JavaScript para Manipular o Foco

O JavaScript fornece alguns métodos para gerenciar o foco dos elementos:

  • element.focus() → Coloca o foco em um elemento.
  • element.blur() → Remove o foco de um elemento.

Exemplo Básico

document.getElementById("meuBotao").focus(); // Coloca o foco no botão automaticamente

Monitorando o Foco com Eventos

Você pode usar eventos para detectar quando um elemento recebe ou perde o foco:

  • focus → Disparado quando um elemento recebe o foco.
  • blur → Disparado quando um elemento perde o foco.
  • focusin e focusout → Semelhantes ao focus e blur, mas propagam o evento para elementos ancestrais.

Exemplo de Evento focus e blur

const input = document.getElementById("meuInput");

input.addEventListener("focus", () => {
console.log("O campo recebeu foco!");
});

input.addEventListener("blur", () => {
console.log("O campo perdeu o foco!");
});

Criando uma Navegação com o Teclado

Podemos capturar eventos de teclado para permitir navegação entre elementos interativos.

Exemplo: Capturando a tecla Tab

document.addEventListener("keydown", function(event) {
if (event.key === "Tab") {
console.log("A tecla Tab foi pressionada!");
}
});

Exemplo: Navegação entre Botões com as Teclas Seta

const botoes = document.querySelectorAll("button"); let indice = 0; document.addEventListener("keydown", function(event) {
if (event.key === "ArrowRight") {
indice = (indice + 1) % botoes.length; // Avança para o próximo botão
} else if (event.key === "ArrowLeft") {
indice = (indice - 1 + botoes.length) % botoes.length; // Volta para o botão anterior
}
botoes[indice].focus();
});

Garantindo Acessibilidade em Modais

Quando um modal é aberto, devemos mover o foco para o modal e impedir que o usuário navegue fora dele.

Exemplo: Foco no Modal

const botaoAbrir = document.getElementById("abrirModal");
const modal = document.getElementById("meuModal");
const botaoFechar = document.getElementById("fecharModal");

botaoAbrir.addEventListener("click", () => {
modal.style.display = "block";
modal.querySelector("button").focus(); // Foca no primeiro botão do modal
});

botaoFechar.addEventListener("click", () => {
modal.style.display = "none";
botaoAbrir.focus(); // Retorna o foco ao botão que abriu o modal
});

Evitando Problemas de Foco

Evitar tabindex Negativo

Não use tabindex="-1" em elementos interativos, pois isso pode impedir a navegação pelo teclado.

Forçar Foco em Elementos Personalizados

Se um elemento interativo for criado dinamicamente, use element.focus() para garantir que ele receba foco.

const novoBotao = document.createElement("button");
novoBotao.textContent = "Novo Botão";
document.body.appendChild(novoBotao);
novoBotao.focus();
