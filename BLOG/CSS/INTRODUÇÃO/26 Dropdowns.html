Dropdowns

Os dropdowns (menus suspensos) são comumente usados em interfaces de usuário para economizar espaço e oferecer uma navegação compacta e interativa. Em CSS, podemos criar menus dropdown simples e responsivos utilizando apenas HTML e CSS, sem necessidade de JavaScript.

Estrutura Básica de um Dropdown

A estrutura de um menu dropdown geralmente é composta por uma lista (ul) com um item (li) que contém outro sub-menu (ul).

<div class="dropdown">
<button class="dropbtn">Menu</button>
<div class="dropdown-content">
<a href="#">Item 1</a>
<a href="#">Item 2</a>
<a href="#">Item 3</a>
</div>
</div>

Explicação:

  • A classe dropdown envolve o botão que abre o menu suspenso.
  • O sub-menu é a div com a classe dropdown-content, que contém os links de navegação.

Estilizando o Dropdown com CSS

CSS é usado para ocultar o menu suspenso e exibí-lo apenas quando o botão for hoverado.

/* O estilo do botão */
.dropbtn {
background-color: #4CAF50;
color: white;
padding: 16px;
font-size: 16px;
border: none;
cursor: pointer;
}

/* Esconde o conteúdo do dropdown por padrão */
.dropdown-content {
display: none;
position: absolute;
background-color: #f9f9f9;
min-width: 160px;
box-shadow: 0px 8px 16px rgba(0, 0, 0, 0.2);
z-index: 1;
}

/* Exibe o conteúdo quando o botão é hoverado */
.dropdown:hover .dropdown-content {
display: block;
}

/* Estilo dos links dentro do dropdown */
.dropdown-content a {
color: black;
padding: 12px 16px;
text-decoration: none;
display: block;
}

/* Muda a cor do link quando o mouse passa por cima */
.dropdown-content a:hover {
background-color: #ddd;
}

Explicação:

A classe dropbtn estiliza o botão do menu.
  • A classe dropdown-content é oculta inicialmente com display: none;.
  • O seletor dropdown:hover .dropdown-content exibe o menu quando o usuário passa o mouse sobre a dropdown.
  • Os links dentro do menu têm o estilo padrão e mudam de cor quando o usuário passa o mouse.

Dropdown Responsivo

A responsividade do dropdown pode ser ajustada para telas menores, garantindo uma boa experiência de navegação em dispositivos móveis.

@media screen and (max-width: 600px) {
.dropdown-content {
position: relative;
box-shadow: none;
}
}

Explicação:

  • O @media ajusta a posição do menu dropdown quando a tela tem menos de 600px de largura.
  • No caso de telas pequenas, o menu suspenso será exibido de forma relativa, sem o efeito de position: absolute; que poderia causar problemas de layout.
