Texto no CSS

O CSS oferece uma variedade de propriedades para estilizar e formatar o texto. A seguir, abordaremos as principais propriedades relacionadas ao texto, incluindo alinhamento, decoração, transformação, espaçamento e sombra.

Alinhamento de Texto (text-align)

A propriedade text-align define o alinhamento horizontal do texto dentro de um elemento de bloco.

Valores Comuns:

  • left: Alinha o texto à esquerda (padrão).
  • right: Alinha o texto à direita.
  • center: Centraliza o texto.
  • justify: Justifica o texto, ajustando o espaçamento entre palavras para que as linhas tenham a mesma largura.

Exemplo:

p {
text-align: justify;
}

Decoração de Texto (text-decoration)

A propriedade text-decoration especifica a decoração adicionada ao texto, como sublinhado, linha sobre o texto ou linha através do texto.

Valores Comuns:

  • none: Sem decoração (padrão).
  • underline: Sublinha o texto.
  • overline: Adiciona uma linha acima do texto.
  • line-through: Adiciona uma linha através do texto (tachado).
  • blink: Faz o texto piscar (não amplamente suportado e geralmente não recomendado).

Exemplo:

a {
text-decoration: underline;
}

Transformação de Texto (text-transform)

A propriedade text-transform controla a capitalização do texto.

Valores Comuns:

  • none: Sem transformação (padrão).
  • capitalize: Converte a primeira letra de cada palavra em maiúscula.
  • uppercase: Converte todo o texto para maiúsculas.
  • lowercase: Converte todo o texto para minúsculas.

Exemplo:

h1 {
text-transform: uppercase;
}

Espaçamento de Texto

O CSS fornece várias propriedades para controlar o espaçamento relacionado ao texto:

Recuo de Texto (text-indent)

Define o recuo da primeira linha de um parágrafo.

Exemplo:

p {
text-indent: 50px;
}

Espaçamento entre Letras (letter-spacing)

Controla o espaço entre caracteres.

Exemplo:

p {
letter-spacing: 2px;
}

Espaçamento entre Palavras (word-spacing)

Controla o espaço entre palavras.

Exemplo:

p {
word-spacing: 5px;
}

Altura da Linha (line-height)

Define a altura da linha, influenciando o espaçamento vertical entre linhas de texto.

Exemplo:

p {
line-height: 1.5;
}

Sombra de Texto (text-shadow)

A propriedade text-shadow adiciona sombra ao texto, permitindo efeitos visuais atraentes.

Sintaxe:

text-shadow: h-shadow v-shadow blur-radius color;

  • h-shadow: Deslocamento horizontal da sombra.
  • v-shadow: Deslocamento vertical da sombra.
  • blur-radius (opcional): Raio de desfoque da sombra.
  • color (opcional): Cor da sombra.

Exemplo:

h1 {
text-shadow: 2px 2px 5px #888888;
}
