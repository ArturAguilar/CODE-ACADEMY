Modelo de Caixa (Box Model)

No CSS, cada elemento é representado como uma caixa retangular composta por várias camadas que determinam seu tamanho e espaçamento em relação a outros elementos. Essas camadas, de dentro para fora, são:

  1. Conteúdo (Content): A área onde o conteúdo (texto, imagens, etc.) é exibido.
  2. Preenchimento (Padding): Espaço entre o conteúdo e a borda.
  3. Borda (Border): A borda ao redor do preenchimento e do conteúdo.
  4. Margem (Margin): Espaço entre a borda do elemento e outros elementos adjacentes.

Propriedades do Modelo de Caixa

Conteúdo (Content)

Define a largura (width) e a altura (height) da área de conteúdo do elemento.

div {
width: 300px;
height: 200px;
}

Preenchimento (Padding)

Controla o espaço interno entre o conteúdo e a borda do elemento.

div {
padding: 20px; /* Aplica 20px de preenchimento em todos os lados */ /* Ou especificar individualmente: */ padding-top: 10px;
padding-right: 15px;
padding-bottom: 10px;
padding-left: 15px;
}

Borda (Border)

Define a borda ao redor do conteúdo e do preenchimento.

div {
border: 5px solid #000; /* Borda de 5px, sólida, cor preta */
/* Ou especificar individualmente: */
border-top: 2px dashed #333;
border-right: 3px dotted #666;
border-bottom: 4px double #999;
border-left: 5px groove #ccc;
}

Margem (Margin)

Controla o espaço externo entre a borda do elemento e outros elementos.

div {
margin: 10px; /* Aplica 10px de margem em todos os lados */
/* Ou especificar individualmente: */
margin-top: 5px;
margin-right: 10px;
margin-bottom: 15px;
margin-left: 20px;
}

Cálculo do Tamanho Total do Elemento

Por padrão, o tamanho total de um elemento é calculado somando-se o conteúdo, o preenchimento, a borda e a margem.

Largura Total:width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Altura Total:height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Exemplo:

div {
width: 300px;
height: 200px;
padding: 10px;
border: 5px solid #000;
margin: 20px;
}

Largura Total:300px (conteúdo) + 20px (preenchimento) + 10px (borda) + 40px (margem) = 370px

Altura Total:200px (conteúdo) + 20px (preenchimento) + 10px (borda) + 40px (margem) = 270px

Ajustando o Modelo de Caixa com box-sizing

A propriedade box-sizing permite alterar a forma como a largura e a altura de um elemento são calculadas.

  • content-box (padrão): A largura e a altura incluem apenas o conteúdo. O preenchimento e a borda são adicionados posteriormente.
  • border-box: A largura e a altura incluem o conteúdo, o preenchimento e a borda.

Exemplo:

div {
box-sizing: border-box;
}

Com box-sizing: border-box;, o tamanho total do elemento permanece conforme especificado, independentemente do preenchimento ou da borda.
