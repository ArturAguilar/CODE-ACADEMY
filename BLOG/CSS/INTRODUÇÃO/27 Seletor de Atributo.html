Seletor de Atributo

Os seletores de atributo no CSS permitem que você estilize elementos com base nos atributos HTML que eles possuem. Isso é muito útil quando queremos aplicar estilos específicos sem precisar adicionar classes ou IDs extras.

Seletor de Atributo Exato ([atributo])

Este seletor estiliza elementos que possuem um atributo específico, independentemente do seu valor.

[input] {
border: 2px solid #d70000;
}

Explicação: Todos os elementos input terão uma borda vermelha de 2px.

Seletor de Atributo com Valor Específico ([atributo="valor"])

Aplica estilos apenas a elementos que possuem um atributo com um valor exato.

[input type="text"] {
background-color: #f0f0f0;
}

Explicação: Somente input type="text" terá um fundo cinza claro.

Seletor de Atributo com Valor Contendo ([atributo*="valor"])

Seleciona elementos cujo valor do atributo contém a sequência especificada.

[class*="btn"] {
padding: 10px;
font-weight: bold;
}

Explicação: Qualquer elemento que tenha btn em sua classe (btn-primary, btn-danger, big-btn, etc.) terá padding de 10px e font-weight: bold.

Seletor de Atributo que Começa com ([atributo^="valor"])

Seleciona elementos cujo valor do atributo começa com um determinado texto.

[href^="https://"] {
color: green;
}

Explicação: Todos os links (a) que começam com "https://" terão a cor verde.

Seletor de Atributo que Termina com ([atributo$="valor"])

Seleciona elementos cujo valor do atributo termina com um determinado texto.

[href$=".pdf"] {
color: red;
}

Explicação: Links (a) que terminam com .pdf (indicando um arquivo PDF) ficarão vermelhos.

Seletor de Atributo que Contém Palavra ([atributo~="valor"])

Seleciona elementos cujo valor do atributo contém uma palavra específica, separada por espaços.

[class~="warning"] {
background-color: yellow;
}

Explicação: Apenas elementos cuja class contém exatamente "warning" (ou seja, class="warning" ou class="alert warning") terão fundo amarelo.

Exemplo Prático

<!DOCTYPE html>
<html lang="pt">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Seletores de Atributo</title>
<style>
/* Input de texto com fundo cinza */
[type="text"] {
background-color: #f0f0f0;
border: 1px solid #ccc;
padding: 8px;
}

/* Links externos em verde */
[href^="https://"] {
color: green;
}

/* Links para PDF em vermelho */
[href$=".pdf"] {
color: red;
}

/* Todos os elementos que possuem "btn" na classe */
[class*="btn"] {
padding: 10px;
font-weight: bold;
}
</style>
</head>
<body>

<input type="text" placeholder="Digite algo...">
<input type="password" placeholder="Senha">

<br><br>

<a href="https://w3schools.com">Link Externo</a> |
<a href="documento.pdf">Baixar PDF</a>

<br><br>

<button class="btn-primary">Botão Principal</button>
<button class="btn-secondary">Botão Secundário</button>

</body>
</html>
