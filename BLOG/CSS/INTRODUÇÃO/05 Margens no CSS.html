Margens no CSS

A propriedade margin define o espaço externo entre um elemento e seus vizinhos.

Sintaxe

elemento {
margin: valor;
}

O valor pode ser em pixels (px), porcentagem (%), auto ou unidades relativas (em, rem, vw, etc.).

Exemplo Simples

HTML

<div class="caixa">Espaçamento externo</div>

CSS

.caixa {
margin: 20px;
background-color: lightblue;
}

Definindo Margens Individualmente

Você pode definir margens separadas para cada lado:

elemento {
margin-top: 10px;
margin-right: 15px;
margin-bottom: 20px;
margin-left: 25px;
}

Ou usar shorthand (atalho):

margin: 10px 15px 20px 25px;
/* topo, direita, baixo, esquerda */

Outras formas:

margin: 10px 20px; /* topo e baixo = 10px, esquerda e direita = 20px */
margin: 15px auto; /* centraliza horizontalmente */

Colapso de Margens

O colapso de margens ocorre quando margens verticais de elementos adjacentes se sobrepõem, ao invés de se somarem.

Como Funciona?

Quando dois elementos têm margens superiores ou inferiores que se encontram, o navegador usa apenas a maior delas, ao invés de somá-las.

Exemplo:

Padding no CSS .caixa1 {
margin-bottom: 30px;
}

.caixa2 {
margin-top: 20px;
}

A margem entre .caixa1 e .caixa2 será 30px, e não 50px (30px + 20px).

Casos em que o Colapso Ocorre

  • Elementos adjacentes (um abaixo do outro).
  • Elemento dentro de um container sem padding ou border.
  • Elemento sem conteúdo com margin-top e margin-bottom.
    •

    Evitando o Colapso

    Adicionar padding ou border no container:

    .container {
    padding: 1px;
    }

    Usar display: flex; ou grid; no container.

    Usar overflow: hidden; no elemento pai.
