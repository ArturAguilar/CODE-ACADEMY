Cores no CSS

Métodos para Definir Cores

Existem quatro formas principais de especificar cores no CSS:

Nomes de Cores

CSS suporta nomes de cores pré-definidos, como:

h1 { color: red; }
p { background-color: lightblue; }

✅ Fácil de usar

❌ Limitado a cores predefinidas

🔹 Exemplo de cores nomeadas: Red, Blue, Green, Yellow, Black, White, Gray, Purple, Cyan, Magenta, etc.

Cores em Hexadecimal (#rrggbb)

Usa um código de 6 caracteres para definir cores:

h1 { color: #ff0000; } /* Vermelho */
p { background-color: #00ff00; } /* Verde */

✅ Mais opções que nomes de cores

✅ Muito usado em design

❌ Menos intuitivo que nomes de cores

Atalho para hexadecimais

  • #ff0000 pode ser abreviado para #f00
  • #00ff00 → #0f0
  • #0000ff → #00f

RGB (rgb(r, g, b))

Define cores usando valores entre 0 e 255 para vermelho, verde e azul:

h1 { color: rgb(255, 0, 0); } /* Vermelho */
p { background-color: rgb(0, 255, 0); } /* Verde */

✅ Fácil de ajustar níveis de cores

✅ Útil para efeitos de transparência (com RGBA)

RGBA (rgba(r, g, b, a))

O "a" no final define a transparência (opacidade), variando de 0 (totalmente transparente) a 1 (totalmente opaco).

h1 { color: rgba(255, 0, 0, 0.5); } /* Vermelho com 50% de transparência */

HSL (hsl(hue, saturation, lightness))

Define cores usando:

  • Hue (Matiz): 0° (vermelho) a 360° (volta completa no espectro)
  • Saturation (Saturação): 0% (sem cor) a 100% (cor total)
  • Lightness (Luminosidade): 0% (preto) a 100% (branco)

h1 { color: hsl(0, 100%, 50%); } /* Vermelho */
p { background-color: hsl(240, 100%, 50%); } /* Azul */

🔹 HSLA adiciona transparência:

h1 { color: hsla(120, 100%, 50%, 0.5); } /* Verde transparente */

Cores Transparentes e Predefinidas

O CSS inclui palavras-chave especiais para transparência:

Nome Descrição
transparent Cor totalmente transparente
currentColor Usa a cor definida no color do elemento
inherit Herda a cor do elemento pai

Exemplo de Transparência:

div {
background-color: transparent;
}

button {
color: currentColor; /* Mantém a cor do texto do elemento pai */
}
