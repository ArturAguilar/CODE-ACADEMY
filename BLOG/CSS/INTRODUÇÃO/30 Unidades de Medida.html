Unidades de Medida

As unidades de medida no CSS definem tamanhos, distâncias e proporções para elementos de um site. Elas podem ser relativas ou absolutas, dependendo do comportamento desejado.

Tipos de Unidades

Tipo Unidades Descrição
Absolutas px, cm, mm, in, pt, pc Fixas, não dependem do contexto
Relativas em, rem, %, vw, vh, vmin, vmax, ex, ch Dependem do elemento pai ou do viewport

Unidades Absolutas

As unidades absolutas são fixas e não se ajustam ao tamanho da tela.

Unidade Equivalência
px (pixels) Unidade mais usada para telas
cm (centímetros) 1 cm = 37.8 px
mm (milímetros) 1 mm = 3.78 px
in (polegadas) 1 in = 2.54 cm = 96 px
pt (pontos) 1 pt = 1/72 in
pc (paicas) 1 pc = 12 pt

Exemplo de Uso

p {
font-size: 16px;
width: 10cm;
height: 1in;
}

Unidades Relativas

As unidades relativas se ajustam dinamicamente ao tamanho da tela ou do elemento pai. Unidades Relativas a Texto

Unidade Baseada em Exemplo
em Tamanho da fonte do elemento pai 2em = 2x o tamanho da fonte
rem Tamanho da fonte raiz (html) 1rem = 16px (padrão do navegador)
ex Altura da letra "x" da fonte Aproximadamente metade de 1em
ch Largura do caractere "0" da fonte atual Útil para definir larguras de caixas de texto

Exemplo de em e rem:

html {
font-size: 16px;
}

p {
font-size: 1.5em; /* 1.5x a fonte do elemento pai */
}

div {
font-size: 2rem; /* 2x a fonte do root (32px) */
}

Unidades Relativas ao Viewport (Tela)
Unidade Baseada em Exemplo
vw 1% da largura da tela 50vw = metade da largura da tela
vh 1% da altura da tela 100vh = altura total da tela
vmin Menor valor entre vw e vh Ajusta ao menor lado da tela
vmax Maior valor entre vw e vh Ajusta ao maior lado da tela

Exemplo de vw e vh:

div {
width: 50vw; /* 50% da largura da tela */
height: 100vh; /* Altura total da tela */
}

Unidades Percentuais (%)

Unidade Baseada em Exemplo
% Tamanho relativo ao elemento pai width: 50% = metade do pai

div {
width: 80%;
}
