Estilizando Tabelas

As tabelas são elementos fundamentais para organizar dados em formato tabular. O CSS oferece diversas propriedades para aprimorar a aparência e a funcionalidade das tabelas.

Exemplo Básico:

<table>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>João</td>
<td>30</td>
</tr>
<tr>
<td>Maria</td>
<td>25</td>
</tr>
</table>

Propriedades CSS para Tabelas

border-collapse

Controle a forma como as bordas das células da tabela são exibidas.

Exemplo:

table {
border-collapse: collapse;
}

Valores:

  • collapse: as bordas das células são combinadas em uma única borda.
  • separate: as bordas das células são separadas.

border-spacing

Defina o espaço entre as células da tabela quando border-collapsefor definido como separate.

Exemplo:

table {
border-spacing: 10px;
}

border

Defina a borda da tabela e das células.

Exemplo:

table, th, td {
border: 1px solid black;
}

padding

Defina o preenchimento interno das células.

Exemplo:

th, td {
padding: 8px;
}

text-align

Defina o alinhamento horizontal do conteúdo das células.

Exemplo:

th, td {
text-align: left;
}

vertical-align

Defina o alinhamento vertical do conteúdo das células.

Exemplo:

th, td {
vertical-align: middle;
}

width e height

Defina a largura e altura das células ou da tabela.

Exemplo:

th, td {
width: 150px;
height: 50px;
}

Tamanho da Tabela

Para definir o tamanho da tabela, você pode usar as propriedades widthe height.

Exemplo:

table {
width: 100%;
height: auto;
}

Neste exemplo, a tabela ocupará 100% da largura disponível e a altura será ajustada automaticamente conforme o conteúdo.

Alinhamento da Tabela

Para centralizar a tabela na página, você pode usar as propriedades margine display.

Exemplo:

table {
margin: 0 auto;
display: block;
}

Isso centraliza a tabela horizontalmente na página.

Estilizando a Tabela

Você pode aplicar estilos adicionais para melhorar a aparência da tabela, como cores de fundo, bordas e efeitos de foco.

Exemplo:

table {
width: 100%;
border-collapse: collapse;
}

th, td {
border: 1px solid #ddd;
padding: 8px;
text-align: left;
}

tr:nth-child(even) {
background-color: #f2f2f2;
}

tr:hover {
background-color: #ddd;
}

Neste exemplo, uma tabela tem bordas finas, células com preenchimento, linhas alternadas com fundo cinza claro e um efeito de pairar que altera o cor de fundo das linhas.

Tabelas Responsivas

Para tornar tabelas responsivas e adaptáveis ​​a diferentes tamanhos de tela, você pode usar uma propriedade overflowem conjunto com max-width.

Exemplo:

.table-container {
max-width: 100%;
overflow-x: auto;
}

table {
width: 100%;
min-width: 600px;
}

Neste exemplo, uma tabela será exibida dentro de um contêiner que permite rolagem horizontal quando a largura da tabela exceder a largura da tela.
