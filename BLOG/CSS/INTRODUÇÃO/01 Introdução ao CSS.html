Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. O CSS descreve como os elementos devem ser renderizados na tela, no papel, na fala ou em outras mídias.

O que é CSS?

CSS é uma linguagem de folhas de estilo que descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias. Ele economiza muito trabalho, pois pode controlar o layout de várias páginas web de uma só vez.

Vantagens do CSS:

  • Economiza tempo
  • Facilita a manutenção
  • Permite múltiplos estilos para uma página
  • Melhora a acessibilidade
  • Permite layouts mais complexos

Sintaxe Básica

A sintaxe CSS consiste em um seletor e um bloco de declaração:

seletor {
propriedade: valor;
}

Exemplo:

p {
color: blue;
text-align: center;
}

Três Formas de Inserir CSS

Método Descrição
CSS Externo O estilo é definido em um arquivo .css separado
CSS Interno O estilo é definido dentro da tag <style> no cabeçalho do documento
CSS Inline O estilo é definido diretamente no elemento usando o atributo style

Seletores CSS

Os seletores são usados para "encontrar" (ou selecionar) os elementos HTML que você deseja estilizar.

Tipos de Seletores:

  • Seletor de elemento (p, h1, div, etc.)
  • Seletor de ID (#id)
  • Seletor de classe (.classe)
  • Seletor universal (*)
  • Seletor de atributo ([atributo])

Propriedades CSS

As propriedades CSS são usadas para definir o estilo dos elementos HTML selecionados.

Exemplos de Propriedades:

  • color: define a cor do texto
  • background-color: define a cor de fundo
  • font-family: define a fonte do texto
  • font-size: define o tamanho da fonte
  • text-align: define o alinhamento do texto
