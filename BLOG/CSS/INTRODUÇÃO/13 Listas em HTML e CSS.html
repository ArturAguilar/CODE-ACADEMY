Listas em HTML e CSS

Em HTML, existem dois tipos principais de listas:

  • Listas Não Ordenadas (ul): Itens de lista marcados com marcadores (geralmente círculos ou quadrados).
  • Listas Ordenadas (ol): Itens da lista numerados ou alfabéticos.

Para definir um item da lista, use a tag li.

Exemplo:

<ul>
<li>Café</li>
<li>Chá</li>
<li>Coca-Cola</li>
</ul>

Propriedades CSS para listas

list-style-type

Defina o tipo de marcador para os itens da lista.

Exemplo:

ul {
list-style-type: square;
}

Neste caso, os itens da lista serão marcados com quadrados.

Valores Comuns:

  • disc: marcador padrão (círculo preenchido).
  • circle: círculo vazio.
  • square: quadrado.
  • decimal: números (1, 2, 3, ...).
  • lower-alpha: letras minúsculas (a, b, c, ...).
  • upper-alpha: letras secretas (A, B, C, ...).

Nota: Alguns valores são específicos para listas ordenadas ou não ordenadas.

list-style-position

Determina a posição do marcador em relação ao conteúdo do item da lista.

Exemplo:

ul {
list-style-position: inside;
}

Valores:

  • outside: o marcador fica fora do conteúdo do item (padrão).
  • inside: o marcador fica dentro do conteúdo do item.

list-style-image

Substitua o marcador padrão por uma imagem.

Exemplo:

ul {
list-style-image: url
('imagem-do-marcador.png');
}

Nota: Se a imagem não puder ser incluída, o marcador padrão será exibido.

list-style

Propriedade abreviada para definir list-style-type, list-style-positione list-style-imageem uma única declaração.

Exemplo:

ul {
list-style: square inside url
('imagem-do-marcador.png');
}

Neste exemplo, os itens da lista terão um marcador quadrado, posicionado dentro do conteúdo do item, e a imagem especificada será usada como marcador.

Removendo Marcadores e Estilos Padrão

Para remover os marcadores e os estilos padrão de listas, você pode usar as seguintes propriedades:

Exemplo:

ul {
list-style-type: none;
margin: 0;
padding: 0;
}

Isso remove os marcadores, margens e preenchimentos padrão, permitindo um controle total sobre a aparência da lista.

Estilizando Listas como Botões

Você pode estilizar listas para que seus itens sejam comportados como botões, utilizando propriedades CSS como display, padding, background-color, entre outras.

Exemplo:

ul {
list-style-type: none;
margin: 0;
padding: 0;
}

ul li {
display: inline-block;
padding: 10px 20px;
background-color: #4CAF50;
color: white;
cursor: pointer;
}

ul li:hover {
background-color: #45a049;
}

Neste exemplo, os itens da lista são exibidos como blocos em linha, com preenchimento, cor de fundo e efeito de foco.
