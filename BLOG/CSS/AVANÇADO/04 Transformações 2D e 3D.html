Transformações 2D e 3D

O CSS3 oferece a capacidade de transformar elementos na página, seja em 2D ou 3D. Isso proporciona efeitos visuais impressionantes, como rotação, deslocamento, dimensionamento e distorção.

Transformações 2D (2D Transforms)

As transformações 2D permitem que você mova, rotacione, redimensione e incline um elemento.

Propriedades Comuns de Transformações 2D

transform

É a propriedade básica usada para aplicar transformações em elementos. Vários tipos de transformação podem ser combinados.

Sintaxe:

transform: tipo_de_transformacao( valor );

Exemplos de Transformações 2D:

rotate() - Rotaciona um elemento.

div {
transform: rotate(45deg); /* Rotaciona 45 graus no sentido horário */
}

scale() - Redimensiona o elemento.

div {
transform: scale(1.5); /* Aumenta o tamanho do elemento em 1.5 vezes */
}

translate() - Desloca o elemento.

div {
transform: translate(50px, 100px); /* Desloca 50px para a direita e 100px para baixo */
}

skew() - Inclina o elemento.

div {
transform: skew(20deg, 10deg); /* Inclina o elemento 20 graus no eixo X e 10 graus no eixo Y */
}

Exemplo de Transformações 2D Combinadas:

div {
transform: rotate(45deg) scale(1.2) translate(50px, 100px);
}

Transformações 3D (3D Transforms)

As transformações 3D permitem manipular os elementos em um espaço tridimensional. A principal diferença é o uso de um eixo adicional (Z), o que permite efeitos de profundidade.

Propriedades Comuns de Transformações 3D

transform

Aqui, as transformações 3D podem ser combinadas da mesma forma que as 2D, mas com o uso do eixo Z.

Exemplos de Transformações 3D:

rotateX() - Rotaciona o elemento no eixo X (horizontal).

div {
transform: rotateX(45deg); /* Rotaciona o elemento 45 graus no eixo X */
}

rotateY() - Rotaciona o elemento no eixo Y (vertical).

div {
transform: rotateY(45deg); /* Rotaciona o elemento 45 graus no eixo Y */
}

rotateY() - Rotaciona o elemento no eixo Y (vertical).

div {
transform: rotateY(45deg); /* Rotaciona o elemento 45 graus no eixo Y */
}

translateZ() - Move o elemento para frente ou para trás ao longo do eixo Z (profundidade).

div {
transform: translateZ(50px); /* Move o elemento 50px para frente */
}

perspective() - Controla o efeito de perspectiva, aplicando uma profundidade em um espaço 3D.

.container {
perspective: 500px; /* Cria uma perspectiva de 500px */
}

div {
transform: rotateY(45deg);
}

Exemplo de Transformações 3D Combinadas:

div {
transform: rotateX(30deg) rotateY(30deg) translateZ(100px);
}

Exemplo Completo de Transformações 2D e 3D

/* Transformação 2D */
div {
width: 200px;
height: 200px;
background-color: lightblue;
transform: rotate(45deg) scale(1.5) translate(50px, 100px);
}


/* Transformação 3D */
.container {
perspective: 500px;
}


div {
width: 200px;
height: 200px;
background-color: lightcoral;
transform: rotateY(45deg) translateZ(100px);
}
