Imagens

O CSS3 oferece várias técnicas para manipular imagens, desde a simples exibição até efeitos avançados.

Trabalhando com Imagens

✅ Como Definir uma Imagem de Fundo

body {
background-image: url('imagem.jpg');
background-size: cover;
background-repeat: no-repeat;
}

  • background-image: Define uma imagem como fundo.
  • background-size: cover: Faz com que a imagem ocupe todo o espaço disponível.
  • background-repeat: no-repeat: Evita a repetição da imagem.

Como Centralizar uma Imagem

✅ Centralizar Imagem Dentro de um Contêiner

.container {
display: flex;
justify-content: center;
align-items: center;
height: 300px;
border: 2px solid red;
}

.container img {
max-width: 100%;
height: auto;
}

  • display: flex: Usa flexbox para facilitar o alinhamento.
  • justify-content: center: Centraliza horizontalmente.
  • align-items: center: Centraliza verticalmente.

✅ Centralizar uma Imagem como Fundo

body {
background: url('imagem.jpg') no-repeat center center;
background-size: cover;
}

  • center center: Mantém a imagem centralizada.
  • background-size: cover: Ajusta o tamanho para cobrir todo o espaço.

Aplicando Filtros em Imagens

✅ Exemplo de Aplicação de Filtros

img {
filter: grayscale(100%);
}

Principais Filtros CSS3:

Filtro Descrição
grayscale(100%) Deixa a imagem em preto e branco.
blur(5px) Aplica um efeito de desfoque.
brightness(200%) Aumenta o brilho.
contrast(150%) Ajusta o contraste.
sepia(100%) Dá um efeito de foto antiga.

Criando Formas com Imagens no CSS

Criando Imagens Redondas

img {
width: 150px;
height: 150px;
border-radius: 50%;
}

  • border-radius: 50%: Torna a imagem circular.

✅ Criando Formas Personalizadas com clip-path

img {
clip-path: polygon(50% 0%, 100% 100%, 0% 100%);
}

  • lip-path: Define uma máscara para recortar a imagem em formatos personalizados.
