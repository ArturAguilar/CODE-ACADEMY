Múltiplas Colunas

O CSS3 permite criar layouts de múltiplas colunas, uma técnica útil para dividir conteúdo de forma mais legível e organizada, como em artigos ou blogs. A seguir, veremos as propriedades relacionadas a colunas no CSS.

Propriedades de Múltiplas Colunas

column-count

Define o número de colunas em que o conteúdo será dividido.

✅ Exemplo:

.columns {
column-count: 3;
}

Isso dividirá o conteúdo da classe .columns em 3 colunas.

column-gap

Especifica o espaçamento entre as colunas.

✅ Exemplo:

.columns {
column-gap: 20px;
}

O espaçamento entre as colunas será de 20px.

column-rule-style

Define o estilo da linha entre as colunas, como sólido, pontilhado, etc.

✅ Exemplo:

.columns {
column-rule-style: dashed;
}

Isso aplicará uma linha pontilhada entre as colunas.

column-rule-width

Especifica a largura da linha entre as colunas.

✅ Exemplo:

.columns {
column-rule-width: 5px;
}

A linha entre as colunas terá 5px de largura.

column-rule-color

Define a cor da linha entre as colunas.

✅ Exemplo:

.columns {
column-rule-color: #d70000;
}

A linha entre as colunas será de cor #d70000 (vermelho).

column-rule

É uma shorthand para combinar column-rule-style, column-rule-width e column-rule-color em uma única propriedade.

✅ Exemplo:

.columns {
column-rule: 3px solid #d70000;
}

A linha entre as colunas será 3px de largura, sólida e com cor #d70000.

column-span

Indica se o conteúdo pode se estender por mais de uma coluna (para elementos como cabeçalhos ou imagens).

✅ Exemplo:

header {
column-span: all;
}

O cabeçalho ocupará todas as colunas.

column-width

Define a largura das colunas. Quando combinado com column-count, o navegador ajusta o número de colunas para se ajustar à largura definida.

✅ Exemplo:

.columns {
column-width: 200px;
}

Cada coluna terá 200px de largura.

Exemplo Completo

Aqui está um exemplo completo de como usar essas propriedades para criar um layout com múltiplas colunas:

✅ Exemplo de Código Completo:

<div class="columns">
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec auctor ipsum eu metus condimentum, sit amet scelerisque elit ultricies. Phasellus eget urna urna.</p>

<p>Morbi nec dictum leo. Aliquam erat volutpat. Integer congue, dui at venenatis pharetra, justo eros facilisis libero, et efficitur risus enim at elit. Curabitur lobortis massa at felis tempor, sit amet convallis eros consequat.</p>
</div>

.columns {
column-count: 3;
column-gap: 20px;
column-rule: 2px solid #d70000;
column-width: 250px;
}

O conteúdo será dividido em 3 colunas, com 20px de espaçamento entre elas e uma linha vermelha separando as colunas.
