Fontes

A propriedade de fontes no CSS3 permite customizar a aparência do texto, tornando-o mais estiloso e ajustado ao design da página. Podemos controlar a família de fontes, peso, estilo, tamanho, e mais, para garantir uma tipografia adequada.

Propriedades de Fontes

font-family

Define a família de fontes para o texto. Podemos especificar várias fontes, caso a primeira não esteja disponível no dispositivo do usuário.

Sintaxe:

font-family: nome_da_fonte, fonte_genérica;

Exemplo:

p {
font-family: 'Arial', sans-serif;
}

font-size

Define o tamanho da fonte. Podemos usar unidades como px, em, rem, %, etc.

Sintaxe:

font-size: tamanho;

Exemplo:

h1 {
font-size: 36px;
}

font-weight

Define o peso (espessura) da fonte. Pode ser um valor numérico ou uma palavra-chave como bold, normal, lighter.

Sintaxe:

font-weight: valor;

Exemplo:

p {
font-weight: bold;
}

font-style

Define o estilo da fonte, como normal, italic (itálico) ou oblique.

Sintaxe:

font-style: estilo;

Exemplo:

i {
font-style: italic;
}

line-height

Define a altura da linha, que afeta o espaçamento entre as linhas de texto.

Sintaxe:

line-height: valor;

Exemplo:

p {
line-height: 1.5;
}

Fontes Personalizadas (Web Fonts)

Você também pode usar fontes personalizadas através do @font-face ou serviços de fontes externas como o Google Fonts.

@font-face

Permite importar uma fonte externa para o seu site.

Sintaxe:

@font-face {
font-family: 'NomeDaFonte';
src: url('fonte.ttf') format('truetype');
}

body {
font-family: 'NomeDaFonte', sans-serif;
}

Google Fonts

Você pode importar fontes diretamente do Google Fonts para o seu projeto, sem precisar fazer o download delas.

Exemplo:

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

<style>
body {
font-family: 'Roboto', sans-serif;
}
</style>

Exemplo Completo de Tipografia

body {
font-family: 'Arial', sans-serif;
font-size: 16px;
font-weight: normal;
line-height: 1.5;
}

h1 {
font-family: 'Georgia', serif;
font-size: 36px;
font-weight: bold;
}

p {
font-size: 14px;
font-style: italic;
}
