Variáveis

As variáveis CSS, também chamadas de CSS Custom Properties, permitem armazenar valores reutilizáveis, tornando o código mais organizado e fácil de manter.

Introdução às Variáveis

Sintaxe básica:

  • Variáveis são definidas dentro de um seletor CSS usando --nome-da-variavel.
  • Para utilizá-las, usamos a função var(--nome-da-variavel).

Exemplo:

:root {
--cor-principal: #d70000;
--tamanho-fonte: 18px;
}

h1 {
color: var(--cor-principal);
font-size: var(--tamanho-fonte);
}

  • O :root define as variáveis globais.
  • O h1 usará --cor-principal como cor e --tamanho-fonte como tamanho da fonte.

Sobrescrevendo Variáveis

As variáveis CSS podem ser sobrescritas em escopos mais específicos.

Exemplo de sobrescrita:

:root {
--cor-texto: black;
}

p {
color: var(--cor-texto);
}

p.destaque {
--cor-texto: red; /* Sobrescrevendo a variável apenas neste escopo */
color: var(--cor-texto);
}

  • O parágrafo com a classe .destaque terá texto vermelho, enquanto os demais permanecerão pretos.

Usando Variáveis CSS no JavaScript

Podemos acessar e modificar variáveis CSS usando JavaScript, permitindo a personalização dinâmica da interface.

Exemplo com JavaScript:

:root {
--cor-de-fundo: lightblue;
}

body {
background-color: var(--cor-de-fundo);
}

document.documentElement.style.setProperty('--cor-de-fundo', 'yellow');

  • O código JavaScript altera a variável --cor-de-fundo para amarelo, mudando a cor do fundo da página dinamicamente.

Usando Variáveis com Media Queries

Podemos ajustar variáveis CSS dinamicamente com Media Queries, tornando o design responsivo mais flexível.

Exemplo de Media Queries:

:root {
--largura-max: 1200px;
--cor-de-fundo: white;
}

@media (max-width: 600px) {
:root {
--cor-de-fundo: lightgray;
}
}

body {
max-width: var(--largura-max);
background-color: var(--cor-de-fundo);
}

  • Para telas menores que 600px, a cor de fundo muda para cinza claro (lightgray).
