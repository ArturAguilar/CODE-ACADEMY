Propriedades Personalizadas

As propriedades personalizadas no CSS, também conhecidas como CSS Custom Properties (variáveis CSS), permitem definir valores reutilizáveis para diversos estilos. Isso facilita a manutenção do código e melhora a organização do projeto.

Definição de Propriedades Personalizadas

Sintaxe:

As propriedades personalizadas são declaradas usando --nome-da-propriedade e acessadas com var(--nome-da-propriedade).

:root {
--cor-principal: #d70000;
--fonte-padrao: Arial, sans-serif;
--tamanho-texto: 16px;
}

body {
background-color: var(--cor-principal);
font-family: var(--fonte-padrao);
font-size: var(--tamanho-texto);
}

  • O :root define variáveis globais, acessíveis em todo o documento.
  • As variáveis são aplicadas dinamicamente nos estilos.

Vantagens de Usar Propriedades Personalizadas

  • ✅ Facilidade de manutenção: Alterar um valor na variável atualiza automaticamente todos os elementos que a utilizam.
  • ✅ Código mais limpo e organizado: Reduz repetições, tornando o CSS mais legível.
  • ✅ Flexibilidade: Permite mudanças dinâmicas com JavaScript ou Media Queries.

Escopo e Sobrescrita de Propriedades

As propriedades podem ser definidas globalmente (:root) ou localmente dentro de um seletor.

:root {
--cor-texto: black;
}

p {
color: var(--cor-texto);
}

p.destaque {
--cor-texto: blue;
color: var(--cor-texto);
}

  • O p.destaque sobrescreve a variável --cor-texto localmente, alterando sua cor para azul.

Uso com JavaScript

Podemos modificar propriedades personalizadas dinamicamente com JavaScript.

:root {
--fundo: lightgray;
}

body {
background-color: var(--fundo);
}

document.documentElement.style.setProperty('--fundo', 'yellow');

  • O código altera --fundo para amarelo, mudando a cor de fundo dinamicamente.

Uso com Media Queries

Propriedades personalizadas também podem ser ajustadas conforme o tamanho da tela.

:root {
--tamanho-fonte: 18px;
}

@media (max-width: 600px) {
:root {
--tamanho-fonte: 14px;
}
}

p {
font-size: var(--tamanho-fonte);
}

Em telas menores que 600px, --tamanho-fonte será 14px, tornando o texto menor para dispositivos móveis.
