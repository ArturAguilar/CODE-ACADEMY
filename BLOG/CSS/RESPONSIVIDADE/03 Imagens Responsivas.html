Imagens Responsivas

Criar imagens responsivas é essencial para garantir que elas se ajustem automaticamente ao tamanho da tela, melhorando a experiência do usuário e evitando cortes ou distorções.

Tornando Imagens Responsivas

A melhor forma de fazer isso é usar max-width: 100% e height: auto, garantindo que a imagem nunca fique maior que o contêiner.

img {
max-width: 100%;
height: auto;
}

Com isso, a imagem reduz de tamanho em telas menores sem perder a proporção original.

Exemplo:

<img src="imagem.jpg" alt="Imagem Responsiva">

Ajustando Imagens com object-fit

A propriedade object-fit permite controlar o encaixe da imagem no contêiner, evitando cortes indesejados.

img {
width: 100%;
height: 300px;
object-fit: cover;
}

  • cover → A imagem preenche o espaço sem distorcer.
  • contain → A imagem se ajusta sem cortar partes.

Exemplo com diferentes valores:

.object-cover {
width: 100%;
height: 300px;
object-fit: cover;
}

.object-contain {
width: 100%;
height: 300px;
object-fit: contain;
}

<img class="object-cover" src="imagem.jpg" alt="Imagem com cover">
<img class="object-contain" src="imagem.jpg" alt="Imagem com contain">

Imagens Diferentes para Cada Dispositivo (picture e srcset)

Podemos usar a tag picture para carregar imagens diferentes dependendo do tamanho da tela.

<picture>
<source srcset="imagem-pequena.jpg" media="(max-width: 600px)">
<source srcset="imagem-media.jpg" media="(max-width: 1024px)">
<img src="imagem-grande.jpg" alt="Imagem Responsiva">
</picture>

Isso economiza dados móveis e melhora a performance em dispositivos pequenos.

Criando um Banner Responsivo

Podemos usar background-image e background-size: cover; para criar banners adaptáveis.

.banner {
background-image: url('banner.jpg');
background-size: cover;
background-position: center;
height: 300px;
}

A imagem sempre se ajusta ao tamanho do banner.
